# client/src の使い方

このフォルダには、クライアント機用の Arduino スケッチを置きます。

## 使い方
1. Arduino IDE でこのフォルダ内の `.ino` ファイルを開きます。
2. ボードを `Arduino UNO R4 WiFi` に設定します。
3. 必要なポートを選んで書き込みます。

## 現在の内容
- `beat_60bpm.ino`: ホスト機から送られてくる信号を受信し、受信状態を内蔵 LED に反映します。

## 配線
- ホスト機の送信ピンとクライアント機の受信ピンを接続します。
- GND をホスト機とクライアント機で共通化します。

## 注意
- このフォルダのコードは受信専用です。
- 受信ピン番号を変える場合は、スケッチ内の定義を修正してください。
