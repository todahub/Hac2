# HowTo: 配線と動作確認

## 必要部品
- Arduino（ホスト、例: Arduino UNO R4 WiFi）
- クライアント機器 x N（各クライアントは電圧検出できる回路/ADC入力を持つこと）
- 可変抵抗（ポテンショメータ）10kΩ
- ジャンパーワイヤー、GND 結線用ワイヤー
- （任意）バッファ用オペアンプまたはフォロワ（各クライアントごとに推奨）

## ピン割り当て（スケッチ準拠）
- ホスト: `POT_PIN` = `A1`（可変抵抗のワイパー）
- ホスト: `DAC_PIN` = `DAC`（コードで `DAC_PIN` に設定されている出力）
- 全体共通: `5V` と `GND` をホストと全クライアントで共有
 - ホスト: `SWITCH_START_PIN` = `D4`（再生開始ボタン、外部プルダウン抵抗を使用）
 - ホスト: `SWITCH_STOP_PIN` = `D5`（再生停止ボタン、外部プルダウン抵抗を使用）

## 配線手順
1. 電源: ホストの `5V` と `GND` を各クライアントの `5V` / `GND` に接続する（共通グランド必須）。
2. 可変抵抗: 可変抵抗の端子の一方を `5V`、もう一方を `GND`、中央のワイパーをホストの `A1`（`POT_PIN`）に接続する。
3. 出力バス: ホストの `DAC` ピン（スケッチの `DAC_PIN`）を各クライアントの検出回路入力に接続する。
	- 直接接続する場合、各クライアントの入力は高インピーダンスにしてください（例: ≥100kΩ）。
	- 複数のクライアントへ駆動する際は、電圧が分割・ロードされないようにホストの DAC 出力にバッファ（オペアンプの電圧フォロワ）を入れることを推奨します。
4. しきい値回路（クライアント側）: クライアントはバス電圧を読み取り、以下のレンジで ID を判定します。確実にデジタル入力化する場合は比較器／コンパレータで閾値を設定してください。

3. タクトスイッチ: 2 個のタクトスイッチを用意します（アクティブHIGH設定）。
	- START スイッチの一端を `5V` に、もう一端をホストの `D4`（`SWITCH_START_PIN`）に接続します。
	- STOP  スイッチの一端を `5V` に、もう一端をホストの `D5`（`SWITCH_STOP_PIN`）に接続します。
	- 安定化のため、各ピンから `10kΩ` のプルダウン抵抗を `GND` に接続してください（外部プルダウン）。
	- スイッチ未押下ではピンは `LOW`、押下で `HIGH` になります（アクティブHIGH）。
	- `START` は押すと再生開始、`STOP` は押すと再生停止します。
	- システム起動時は停止状態（`Playing=OFF`）から始まります。

## Valutage（レンジ）
- ID1: 約 0.51 V 〜 1.50 V
- ID2: 約 1.51 V 〜 2.50 V
- ID3: 約 2.51 V 〜 3.50 V
- ID4: 約 3.51 V 〜 4.50 V

（ホスト側スケッチ `MID_VOLTAGES` は各 ID の中点電圧を出力します。）

## クライアント側簡易検出例（擬似コード）
```cpp
int raw = analogRead(A1); // クライアントの検出ADC
float v = raw * 5.0 / 1023.0;
if (v >= 0.51 && v <= 1.50) id = 1;
else if (v >= 1.51 && v <= 2.50) id = 2;
else if (v >= 2.51 && v <= 3.50) id = 3;
else if (v >= 3.51 && v <= 4.50) id = 4;
else id = 0; // 未割り当て
```

## 注意事項 / 推奨
- DAC から複数デバイスへ直結する場合、電圧ドロップや負荷で電圧が変わる可能性があります。各クライアントに高入力インピーダンス（またはバッファ）を使用してください。
- ノイズ対策として、バスに 0.1uF のデカップリングコンデンサを追加すると安定します。
- クライアント側は ADC の参照電圧や分解能に依存するため、実装前に実測でしきい値を調整してください。

## 動作確認手順
1. すべての配線を接続する（電源は最後に入れる）。
2. ホストスケッチをアップロードし、シリアルモニタ（115200bps）を開く。
3. ポテンショメータを回して BPM 値（シリアル出力）と出力電圧を確認する。
4. クライアントが期待どおり ID を検出するか確認する。
5. タクトスイッチを短押しして再生/停止が切り替わることを確認する。シリアルに `Playing=ON`/`Playing=OFF` が表示されます。
6. 発信確認用 LED: 発信と同時に内蔵 LED が点灯します。ID1 のときは 1 回点灯、ID2〜4 のときは 2〜4 回高速点滅します。
