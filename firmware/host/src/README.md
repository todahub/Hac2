# host/src の使い方

このフォルダには、ホスト機用の Arduino スケッチを置きます。

## 使い方
1. Arduino IDE でこのフォルダ内の `.ino` ファイルを開きます。
2. ボードを `Arduino UNO R4 WiFi` に設定します。
3. 必要なポートを選んで書き込みます。

## 現在の内容
- `beat_60bpm.ino`: 60 BPM でパルス信号を送出します。

## 配線
- ホスト機の送信ピンをクライアント機の受信ピンへ接続します。
- GND をクライアント機と共通にします。

## 注意
- 現在の送信ピンはスケッチ内の `SIGNAL_PIN` で定義しています。
- 送信周期やパルス幅を変える場合は、定数を修正してください。
