## AI概要

Frduino（フロドゥイーノ）〜カエルの歌輪唱システム〜の開発用リポジトリです。  
Obsidian の記録をもとに、設計資料・実装・素材を分離した初期構成にしています。

### ディレクトリの役割
- `docs/`: 概要、計画、設計、発表資料
- `hardware/`: 部品一覧、配線、図面
- `firmware/`: Arduino のホスト機・クライアント機
- `processing/`: 音生成と再生処理
- `assets/`: 画像、SVG、drawio
