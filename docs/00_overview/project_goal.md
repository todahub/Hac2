## AI概要

### プロジェクトの目的
- ITや情報技術に詳しくない人でも、直感的に参加できる合奏・輪唱体験を提供する。
- 音楽と視覚演出を組み合わせて、親しみやすいエンターテインメント作品にする。

### 解決したい課題
- DAWやデジタル楽器は操作が難しく、子どもや高齢者にはハードルが高い。
- 画面中心の操作は、演奏への参加感が伝わりにくい。
- 複数端末の同期演奏では、通信順序によるズレが目立ちやすい。

### 独自性
- ホスト機がテンポと拍を管理し、ブロードキャストで全端末へ同時に伝える。
- Processing で音を生成し、カエルの造形で演奏状態を視覚化する。
