## AI概要

- **プロジェクト名**: Frduino（フロドゥイーノ）〜カエルの歌輪唱システム〜
- **目的**: 子どもやお年寄りなど、ITに詳しくない人でも視覚と聴覚で楽しめる演奏体験を作る。
- **最終目標**: 「カエルの合唱」の輪唱として、違和感なく楽しめる同期演奏システムを構築する。

### システムの主な構成
- ホスト機のArduinoが拍と開始を管理し、複数のクライアント機へ情報を送る。
- 楽器音の生成と再生には Processing を使う。
- カエルの見た目や口の開閉などで、視覚的にもわかりやすくする。

### 技術的特徴
- BPMを演奏中でも変更できる。
- ブロードキャスト通信で同期ズレと通信遅延を抑える。
- Arduinoの並列通信制約を前提に、全クライアントへ同じ拍情報を送る。
