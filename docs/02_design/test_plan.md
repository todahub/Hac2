## AI概要

### 単体テスト
- 制御データの受信成功率を 100% に近づける。
- 不正な長さのパケットを検出できることを確認する。
- 自機以外の演奏指示を誤って受けないことを確認する。
- 内部遅延が 1ms 以内に収まることを確認する。

### 結合テスト
- ホスト機からクライアント機へ拍情報が正しく伝わるか確認する。
- テンポ変更が全端末へ反映されるか確認する。
- Processing で生成した音が Arduino 側の再生と一致するか確認する。

### システムテスト
- 音色らしさを評価する。
- エンタメ性を評価する。
- 同期ズレの最大値が 30ms 以内に収まることを目標にする。
- ノイズ低減や波形品質を確認する。

### 評価指標
- 定性的: 音色、面白さ、視覚的わかりやすさ
- 定量的: PSNR、同期ズレ、遅延、波形立ち上がり時間
