## AI概要

### FBS
| 機能 | 内容 |
|---|---|
| 1 | カエルの人形による演奏を見て楽しめる |
| 2 | 複数の楽器の音を聴ける |
| 3 | 輪唱として音楽を楽しめる |
| 4 | テンポの変化を楽しめる |
| 5 | 楽器同士が連携して演奏できる |
| 6 | 電子システムとして音を再生できる |

### PBS
| 部品 | 内容 |
|---|---|
| 1 | 演奏表示部 |
| 2 | 楽器音再生部 |
| 3 | 輪唱演奏制御部 |
| 4 | テンポ操作部 |
| 5 | 通信連携部 |
| 6 | 音生成・再生基盤 |

### 対応の要点
- 表示部はカエルの造形と動作表現を担当する。
- 再生部はドラムと各楽器の音を担当する。
- 制御部は輪唱の開始順とテンポ追従を担当する。
- 通信部は BPM と拍情報の送受信を担当する。
