# processing/src の使い方

このフォルダには、Processing で動かす処理や可視化用のスケッチを置きます。

## 使い方
1. Processing を起動します。
2. このフォルダに置いた `.pde` もしくは関連ファイルを開きます。
3. 必要に応じてシリアル通信や可視化の設定を行います。
